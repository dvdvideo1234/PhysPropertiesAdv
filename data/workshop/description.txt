[h1] Description [/h1]
This script extends your physical properties list by given number of categories

[h1] Feature List [/h1]
[list]
 [*] [url=https://github.com/dvdvideo1234/PhysPropertiesAdv/tree/master/data/physprop_adv/materials]Extendable physical properties list[/url]. Not static like the default in-game one
 [*] Physical properties categories are extracted from file names
 [*] Physical properties names are are extracted from category files using word wrap
[/list]

For detailed manual, please refer to Readme.md

Readme: https://github.com/dvdvideo1234/PhysPropertiesAdv/blob/master/readme.md

Git repository: https://github.com/dvdvideo1234/PhysPropertiesAdv

Report errors to: https://github.com/dvdvideo1234/PhysPropertiesAdv/issues

To install materials
 [1] Create the folder garrysmod\data\physprop_adv\materials
 [2] Populate it with the [url=https://github.com/dvdvideo1234/PhysPropertiesAdv/tree/master/data/physprop_adv/materials]data found here[/url]
