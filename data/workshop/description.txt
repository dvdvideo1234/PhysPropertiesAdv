[h1] Description [/h1]
This script extends your physical properties list by given number of categories

[h1] Feature List [/h1]
[list]
 [*] [url=https://github.com/dvdvideo1234/PhysPropertiesAdv/tree/master/data/physprop_adv/materials]Extendable physical properties list[/url]. Not static like the default in-game one
 [*] Physical properties categories are extracted from file names
 [*] Physical properties names are are extracted from category files using word wrap
 [*] Property copy/cache ( Right click )
 [*] Property apply on a entity ( LeftClick )
 [*] Property apply on a contraption ( Shift + LeftClick )
 [*] Property apply copied/cached ( Use + LeftClick )
 [*] Property apply copied/cached on contraption ( Use + Shift + LeftClick )
 [*] Property apply/restore on all the bones control panel option
 [*] Property restore on a bone ( Reload )
 [*] Property restore on a contraption ( Reload + Shift ) 
[/list]
