[h1] Description [/h1]
This script extends your physical properties list by given number of categories

[h1] Feature List [/h1]
[list]
 [*] Client-side diedicated custom properties and material grouping
 [*] Physical properties (advanced) tool for surface behavior manipulation
 [*] Material (advanced) tool for extended surface texture manipulation
 [*] Physical properties and material tools are fully easily  translatabe
 [*] Extendable [url=https://github.com/dvdvideo1234/PhysPropertiesAdv/tree/master/data/physprop_adv/materials]physical property[/url] and [url=https://github.com/dvdvideo1234/PhysPropertiesAdv/tree/master/data/material_adv/materials]material[/url] lists
 [*] Physical properties and material categories are extracted from file names
 [*] Physical properties and material names are extracted from category files using word wrap
 [*] Property copy/cache ( Right click )
 [*] Property apply on a entity ( LeftClick )
 [*] Property apply on a contraption ( Shift + LeftClick )
 [*] Property apply copied/cached ( Use + LeftClick )
 [*] Property apply copied/cached on contraption ( Use + Shift + LeftClick )
 [*] Property apply/restore on all the bones control panel option
 [*] Property restore on a bone ( Reload )
 [*] Property restore on a contraption ( Reload + Shift )
 [*] Material custom user defined lists can be created
 [*] Material randomize on tool primary fire using the current list
 [*] Material selection list is dynamic and recreatable
[/list]
